# Virtual Environments

```
$ pip install virtualenv
```

```
$ virtualenv python37 # Ejemplo
$ virtualenv --python=/path/to/python2.7 python27 # Ejemplo
```

Se puede tener todos los ambientes en la misma carpeta : `C:\python-envs\`

Activar ambiente:

```
$ C:\python-env\pytonvritual\Scripts>activate
```

Desactivar ambiente:

```
$ C:\python-env\pytonvritual\Scripts>deactivate
```
